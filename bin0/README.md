# Bin0

The binary seed for bootstrapping

It is meant to be the smallest feasible self-hosted binary. The supported language is target-dependent

## Supported Targets

- armv7l-linux
- aarch64-linux
- powerpc-linux
- powerpc64-linux
- powerpc64le-linux
- riscv32-linux
- riscv64-linux
- x86_64-linux
- i686-linux
